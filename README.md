# Charming (Echarts )& Leptos Axum SSR Example with Resize & Reactive
This uses Leptos SSR Template
[Leptos](https://github.com/leptos-rs/leptos) web framework and the [cargo-leptos](https://github.com/akesson/cargo-leptos) tool using [Axum](https://github.com/tokio-rs/axum).

## Running
```bash
cargo install cargo-leptos --locked
```
Clone this repo and run

```bash
cargo leptos watch
```
